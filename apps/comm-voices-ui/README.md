# Workato Community Voices UI

Static frontend for the `workato-comm-voices` MCP demo.

## Scripts

```bash
npm install
npm run dev
npm run build
```

## Local development

```bash
npm run dev
```

The app fetches community posts directly from:

- `https://workato-comm-voices.fly.dev/community-posts`

Build output is generated in `dist/` and can be deployed as static files under `/workato-comm-voices/`.
